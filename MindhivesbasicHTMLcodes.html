In this text <.x> is replaced with (.x)
  • (b), (i) and (u) are style tags.
  • (html) & (/html) in between means it's HTML code.
  • Between the (html) you put (head) - (/head) and (body) - (/body).
  • (head) will contain underlying code.
  • (body) will contain visible context.
  • (meta name="viewport" content="width=device-width, initial-scale=1"), this code will make everything on the page fit on any device's screen.
  • (title) gives the webpage a title.
  • (h1), (h2), ... to make text larger.
  • (p) creates a paragraph.
  • (br) text goes one row down.
  • &-e-n-s-p-; and &-e-m-s-p-; create a gap.
  • (hr) creates a line.
  • Link tag: (a href="x" target="_blank")
    • a makes it clickable
    • x is the URL
    • target="_blank" makes the webpage open in a new tab.
  • Image: (img scr="xb" border="c"), x is the source to the image or the file name, b is the size (e.g., 100x200), c is de size of the border.
  • List: (ul) (li)x(/li)... (/ul), (ul -> ol) makes it numbered.
  • Table: (table border"x" cellpadding="b" cellspacing="c") (tr) (/tr) (/table)
    • x is the size of the border
    • b is the space in the table
    • c is the size of the lines in the table
    • tr is a new row
Tip: If I missed any basics. Just look up "HTML basics" on Youtube.
